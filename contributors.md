# Contributors

The Troupe 513 website is made possible by volunteers who contribute their time, technical skills, and knowledge of the Westerville South theatre community. Thank you to everyone who has helped keep the site useful, welcoming, and up to date.

## Project Contributors

### [Desmond Ducy](https://github.com/DesmoDeuce)

Created and developed the site’s core structure, templates, styling, and content. Desmond has also maintained show information, FAQs, team and board pages, sponsor listings, ticket links, and site imagery.

### [Michael Ducy](https://github.com/mfdii)

Set up and refined the GitHub Actions deployment process, production build configuration, GitHub Pages publishing, and custom-domain support.

### [Ryan Battles](https://github.com/ryanbattles)

Improved project documentation, contributor guidance, local Hugo setup instructions, and deployment validation.

### Troupe 513 Theatre Boosters

Established the repository and continues to support the site’s mission: sharing news, resources, productions, and opportunities from the Westerville South theatre program.

## Additional Thanks

We also appreciate the students, families, educators, creative team members, board members, sponsors, photographers, and volunteers whose information and work appear throughout the site. Their contributions make both the theatre program and this website possible.

## Recognition

This list is based on the repository’s Git history and may not capture every contribution made outside GitHub. If someone is missing or a description needs updating, please open a pull request with the correction.
